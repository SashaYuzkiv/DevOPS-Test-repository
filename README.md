# GitHub

## **Що таке GitHub?**

**GitHub** — це найбільша у світі *платформа* для хостингу IT-проєктів та спільної розробки, що базується на системі контролю версій **Git**.

### **Переваги**

* **Версіонування:** Можливість відслідковувати *кожну зміну* в коді.
* **Співпраця:** Інструменти для командної роботи (Pull Requests).
* **CI/CD:** Інтеграція з інструментами автоматизації (GitHub Actions).

---

## **Команди Git**

| Команда | Опис Дії | Призначення |
| :--- | :--- | :--- |
| `git clone` | Копіює репозиторій на локальну машину. | Початок роботи. |
| `git add .` | **Додає** змінені файли для коміту. | Підготовка змін. |
| `git commit -m "..."` | **Фіксує** зміни в локальній історії. | Збереження версії. |
| `git push` | Відправляє локальні коміти на **GitHub**. | Оновлення віддаленого репозиторію. |
